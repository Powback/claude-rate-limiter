Done! Added a static `fromJSON(str)` method to reconstruct TodoList instances from JSON. It parses the string and restores both the `todos` array and `nextId` counter.

```javascript
static fromJSON(str) {
  const data = JSON.parse(str);
  const instance = new TodoList();
  instance.todos = data.todos;
  instance.nextId = data.nextId;
  return instance;
}
```

Now you can serialize and deserialize:
```javascript
const list = new TodoList();
list.add('Buy milk', 'high');
const json = list.toJSON();
const restored = TodoList.fromJSON(json);
```
